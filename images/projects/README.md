# Project Images

This folder is for your project thumbnails. To add your project images:

1. Add your project images to this folder
2. Name them project-1.jpg, project-2.jpg, project-3.jpg (or .png)
3. Update the HTML in index.html to use your actual project images:
   - Find the project-placeholder divs
   - Replace them with img tags pointing to your images:
     ```html
     <img src="images/projects/project-1.jpg" alt="Project Title">
     ```
   - Remove the project-placeholder divs

The project section will automatically display your images with the same styling as the certificates section.
